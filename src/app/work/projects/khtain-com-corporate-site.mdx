## Overview

Khtain.com is the flagship corporate website for Khtain Block Technology Ltd — a premium digital presence built with Next.js 16 and Once UI. The site features a fully bilingual (English / 中文) architecture with dynamic content switching, GSAP-powered fluid animations, and a strict Glassmorphism design language. It serves as both a brand showcase and a lead-generation platform for two core business divisions: AI Search Visibility and Khtain Labs.

## Key Features

- **Bilingual Architecture**: Complete EN/ZH content parity with seamless language switching. All copy, navigation, and metadata adapt to the selected locale without page reload.
- **GSAP Fluid Animations**: Scroll-triggered reveal sequences, parallax depth effects, and smooth page transitions create a cinematic browsing experience while maintaining 60fps performance.
- **AI Search Visibility**: Marketing landing page for AI-powered GEO/NEO search optimization services, targeting businesses looking to improve their AI search engine presence.
- **Khtain Labs Division**: Showcases the R&D arm of the company, featuring Web3, AI, and infrastructure innovation projects.
- **Responsive Design**: Fully responsive from mobile (320px) to ultra-wide displays, with adaptive typography and layout breakpoints.

## Tech Stack

- **Frontend**: Next.js 16, React 19, TypeScript, TailwindCSS 4
- **Animation**: GSAP ScrollTrigger, Framer Motion
- **Design System**: Once UI with custom Glassmorphism tokens
- **Content**: Markdown-based with gray-matter parsing
- **Deployment**: Vercel (automatic CI/CD from GitHub)

## Deployment

Deployed on Vercel with automatic preview deployments on pull requests. Production domain: `khtain.com`. Edge caching for static assets. Custom OG image generation via `next/og`.

## Outcome

The site serves as the primary digital storefront for Khtain Block Technology, consolidating what were previously separate project pages into one cohesive brand experience. The bilingual architecture supports both North American and Chinese-market clients.
