## Overview

FFT Invoice Helper is a specialized invoicing system for solar installation companies. Unlike generic invoicing tools, it understands solar-specific billing models: per-watt rates, system size tiers, equipment markups, labor line items, permit fees, and government rebate line items. The system generates professional PDF invoices styled with company branding and dispatches them directly to clients via HTML email.

## Key Features

- **Solar-Specific Rate Engine**: Calculate invoice amounts based on installed system wattage with tiered pricing. Automatic equipment cost breakdown from BOM data. Support for per-panel and per-inverter line items.
- **PDF Generation**: WeasyPrint-based PDF rendering with customizable HTML/CSS templates. Includes company logo, project details, payment terms, and itemized line items.
- **Direct Email Dispatch**: HTML email delivery via aiosmtplib. Emails include embedded invoice preview and PDF attachment. Delivery status tracking and resend capability.
- **Client Management**: Track clients with payment history, outstanding balances, and invoice archives. Quick-access dashboard shows accounts receivable aging.
- **Dynamic Calculations**: Real-time invoice preview as rates and quantities are adjusted. Automatic tax calculation (GST/HST) based on province.
- **Multi-Currency**: Support for CAD and USD with configurable exchange rates.

## Tech Stack

- **Frontend**: React 18, Vite, TailwindCSS
- **Backend**: FastAPI, SQLAlchemy, Pydantic
- **Database**: PostgreSQL 15
- **PDF Engine**: WeasyPrint with custom HTML/CSS templates
- **Email**: aiosmtplib (async SMTP)
- **Infrastructure**: Docker Compose (3 services: frontend, backend, database)

## Deployment

Docker Compose on VPS at `invoicehelper.khtain.com`. Three-service architecture ensures separation of concerns: frontend serves the React SPA, backend handles PDF generation and email, and PostgreSQL persists client and invoice data.

## Outcome

The system eliminated manual invoice creation in Excel, reducing per-invoice time from ~20 minutes to under 2 minutes. PDF generation quality matches professional accounting software standards. Clients receive branded, itemized invoices directly in their inbox.
