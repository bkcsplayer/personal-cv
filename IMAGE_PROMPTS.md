## 四、使用说明

1. 把每个 prompt 粘贴到 OpenAI Image Generation（GPT-4o with image）
2. 如果单次生成不理想，加 `--ar 16:9` 或 `--style raw` 调整
3. 生成后按"输出文件"路径保存到 `public/images/` 目录
4. 后续我帮你更新每个项目 MDX 的 `images:` 字段指向新图片
